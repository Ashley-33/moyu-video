# 摸鱼视频（Chrome 扩展）

在任意网页上「伪装式」观看视频：把视频覆盖到页面已有的图片/视频占位元素上，看起来像是页面原生内容。核心场景是办公摸鱼，隐蔽性 > 观看体验。

## 当前版本：v0.5.0

> ⚠️ B站官方 `player.bilibili.com` 已**禁止站外直接播放**，长视频平台（优酷/爱奇艺/芒果等）有 DRM+会员墙且属版权正片，均不支持。本扩展只做 **明文 mp4、无 DRM** 的源：**B站 + 小红书 + 红果短剧**。

### 支持的源
- **B站（稳定）**：普通投稿 `bilibili.com/video/BV...`（含查询参数、分 P、`b23.tv` 短链、裸 BV 号）→ `view` 拿 cid → `playurl` 拿 mp4 直链（匿名约 720P）。用 `declarativeNetRequest` 只给 `*.bilivideo.com` 补 Referer 绕防盗链。
- **B站番剧/影视**：`bangumi/play/ep...` → 免费番剧完整播放；付费内容给"仅试看、暂不支持"提示。
- **小红书（实验性）**：`fetch` 笔记页 → 从 `__INITIAL_STATE__` 抠 `masterUrl`（单段 mp4）→ `http` 转 `https` 播放；偶发风控自动重试 3 次。支持 `xhslink.com` 短链。
- **红果短剧（字节，免费短剧）**：`hongguoduanju.com/player/...` → fetch 播放页抠 `main_url`（明文 https mp4、无 DRM、无防盗链），自动重试 3 次。

### 功能
- **手动拾取**：悬停高亮 `<video>`/`<img>`/≥200×150 块级容器，点击选中，`Esc` 取消。
- **覆盖 & 位置同步**：`position:fixed` 覆盖（不删原元素），rAF + ResizeObserver 跟随滚动/缩放，Shadow DOM 隔离样式；SPA 重建后按选择器路径自动重定位。
- **原生播放**：插件自有 `player.html`（WAR）内用 `<video>` 播直链，无第三方 logo/UI，更隐蔽。
- **老板键**：`Alt+Shift+H`（Mac 为 `⌘+Shift+H`）瞬时藏视频+暂停+静音，再按恢复（保持静音）；可在 `chrome://extensions/shortcuts` 改键。
- **声音安全**：默认静音起播；popup 一键静音 + 音量滑块；音量/透明度记忆到 `chrome.storage.local`。
- **透明度调节** 50%–100%。
- **hover 极简控制条**：鼠标移入才现（播放/暂停、进度、时间、静音、音量）。
- **更换链接**：覆盖播放中直接粘新链接「更换」，位置不变、免还原重选。
- **自动播下一条**：当前视频播完自动接力——红果自动下一集、B站分P 自动下一 P（小红书/单视频/番剧无下一条）；popup 开关，默认开。受页面可暴露的剧集范围限制，超长红果剧可能追到某集停下。
- **popup 引导页**：在受限页面展示"这是什么 / 三步上手 / 小技巧"。

### 尚未实现（M3 剩余）
- **自动寻找占位**：扫描首屏面积最大的 `<video>`/`<img>` 直接覆盖。
- **刷新后状态恢复**：同一 tab 刷新后询问是否恢复。

## 权限
- 常驻：`activeTab` + `scripting` + `storage` + `declarativeNetRequest`
- 动态申请（`optional_host_permissions`，首次用对应源时弹窗）：`b23.tv` / `*.bilibili.com` / `*.bilivideo.com`（B站），`*.xiaohongshu.com` / `xhslink.com` / `*.xhscdn.com`（小红书）

## 加载 / 打包
- **加载测试**：`chrome://extensions/` → 开发者模式 →「加载已解压的扩展程序」→ 选本目录。改过 `content/` 或 `manifest` 后需重载扩展 + 测试网页关掉重开。
- **打包**：`./pack.sh` → 生成 `moyu-video-<版本>.zip`（自动排除文档/构建产物）。
- **上架说明**：见 [STORE.md](STORE.md)（含审核风险评估——盗链解析大概率被拒）。

## 使用步骤
1. 打开一个有图片的普通网页
2. 点扩展图标 → 粘贴 B站/小红书链接 → 「开始」→ 首次弹窗授权对应站点
3. 回网页悬停点选一张图 → 视频（静音）盖上去，跟随滚动
4. 鼠标移上去有控制条；`Alt+Shift+H`（Mac `⌘⇧H`）老板键一键藏/显
5. 图标 →「还原页面」/「更换链接」/ 调音量透明度

## 已知边界
- B站匿名清晰度上限约 720P，大会员/付费番剧取不到；durl 多段视频只播第一段。
- 小红书为实验性，站点改版/风控可能失效；图文笔记无视频。
- 直链带签名会过期，长时间后需重新「开始」。
- 目标元素在 iframe 内部、`chrome://` 等受限页：不支持。
- 页面刷新后覆盖层消失（状态恢复待 M3）。
