# 摸鱼视频 🐟

一个办公摸鱼小插件:把视频**伪装**盖在网页里的图片位置上,看起来像是页面自带的内容,老板走过也不慌。

## 能干嘛

- 把 **B站 / 小红书 / 红果短剧** 的视频,盖到当前网页的图片/视频位置上偷偷看
- **老板键**:一键藏视频、页面秒变正常
- 默认**静音**起播、可调**透明度**,隐蔽第一
- 追剧**自动下一集**(红果剧集 / B站分P)
- 页面**刷新后能恢复**,接着看

## 怎么安装

### 第 1 步:下载

- 打开仓库 <https://github.com/Ashley-33/moyu-video> → 点绿色 **Code** 按钮 → **Download ZIP** → 解压,得到一个 `moyu-video` 文件夹
- (或者别人直接发你 `moyu-video` 的压缩包,解压出来即可)

### 第 2 步:加载到 Chrome

1. 地址栏输入 `chrome://extensions` 回车
2. 打开右上角的 **「开发者模式」**
3. 点 **「加载已解压的扩展程序」**,选中刚解压出的 `moyu-video` 文件夹
4. 列表里出现「摸鱼视频」,就装好了 ✅

### 第 3 步:把图标钉到工具栏

点浏览器右上角的**拼图图标 🧩** → 找到「摸鱼视频」→ 点它右边的**图钉 📌**,那个「摸」图标就固定在工具栏上了。

## 怎么用

1. 打开一个普通网页(新闻 / 知乎 / 微博…都行)
2. 点工具栏上的 **「摸」** 图标
3. 粘一个 **B站 / 小红书 / 红果** 的视频链接,点 **「开始」**
4. 回到网页,鼠标悬停会出现**小手 + “就盖这里”**,点一张图
5. 视频就盖上去了,跟着页面滚动,默认静音
6. 想听声音:鼠标移到视频上,点控制条左下角的**喇叭**
7. **老板键 `Alt+Shift+H`**(Mac 是 `⌘+Shift+H`):一键藏 / 显视频
8. 再点「摸」图标,可以调**音量 / 透明度**、**换链接**、或**还原页面**

## 小提醒

- 第一次用某个平台,会弹一次权限授权,点**允许**就行
- 付费大会员内容、优酷/爱奇艺/芒果这类长视频平台**看不了**(有加密),抖音也**不支持**
- 仅供个人摸鱼娱乐 😉
- 自己改过代码后:回 `chrome://extensions` 点插件的刷新 ↻
